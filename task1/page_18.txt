xmlns="http://www.w3.org/1999/xhtml"><p>Привет, сегодня поговорим, как установить ChromeDriver без костылей и ошибок.</p><p>Если автоматическая загрузка актуальной версии chromedriver (или драйвера для другого браузера) не предусмотрена. Или не работает, и мы при запуске UI-тестов получаем ошибку о том:</p><p>1.      Что у нас отсутствует chromedriver</p><p>2.      Не совпадает версия chromedriver и chrome (также может относиться к любому другому браузеру и его драйверу)</p><p>Скачиваем соответствующую версию chromedriver - <a href="https://googlechromelabs.github.io/chrome-for-testing/#stable">https://googlechromelabs.github.io/chrome-for-testing/#stable</a></p><p>На <strong>Linux</strong></p><ol><li><p>Распаковываем скаченный архив</p></li><li><p>Переходим в каталог /usr/bin</p></li><li><p>Удаляем старую версию chromedriver</p><figure class="full-width "><img src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/138/938/9a4/1389389a4f0f456ae7a84235df45d2d0.png" width="940" height="273" data-src="https://habrastorage.org/getpro/habr/upload_files/138/938/9a4/1389389a4f0f456ae7a84235df45d2d0.png"/></figure></li></ol><div><div class="table"><table><tbody><tr><td data-colwidth="288" width="288"><p align="left">sudo rm chromedriver  </p></td></tr></tbody></table></div></div><ul><li><p>Перемещаем в /usr/bin скаченный и распакованный новый chromedriver</p></li></ul><div><div class="table"><table><tbody><tr><td><p align="left">sudo mv Downloads/chromedriver-linux64/chromedriver /usr/bin/  </p></td></tr></tbody></table></div></div><p>На <strong>Windows</strong></p><ol><li><p>Распаковываем скаченный архив</p></li><li><p>Перемещаем chromedriver в желаемую папку</p></li><li><p>Прописываем системный путь к ней:</p></li></ol><h3>Step #4: Add ChromeDriver to System Path (Windows) </h3><ul><li><p>Right-click your desktop’s “This PC” or “My Computer” icon.</p></li><li><p>Select “<strong>Properties</strong>” from the context menu.</p></li><li><p>Click on “<strong>Advanced system settings</strong>” on the left sidebar.</p></li><li><p>In the System Properties window, click the “<strong>Environment Variables</strong>” button.</p></li><li><p>In the “<strong>System Variables”</strong> section, scroll down and find the <strong>“Path”</strong> variable**.**</p></li><li><p><strong>Click</strong> “<strong>Edit</strong>” or “New” to add a new entry.</p></li><li><p>Enter the path to the folder where you saved the ChromeDriver executable file (e.g., C:\path\to\chromedriver).</p></li><li><p>Click “<strong>OK</strong>” to save the changes.</p></li></ul><h3>Step #5: Verify ChromeDriver Installation </h3><ul><li><p>Open a command prompt or terminal window.</p></li><li><p>Type the following command and press Enter.</p></li></ul><div><div class="table"><table><tbody><tr><td><p align="left">chromedriver --version  </p></td></tr></tbody></table></div></div><p>Удачных тестов!  </p><p></p></div></div></div><!----><!----></div><!----><!----></div><!--]--><!----><div
