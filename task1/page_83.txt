<p><b>Вучедольская культура</b>&#160;— <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D1%8B_%D1%8D%D0%BF%D0%BE%D1%85%D0%B8_%D0%B1%D1%80%D0%BE%D0%BD%D0%B7%D1%8B" title="Балканы эпохи бронзы">балканская</a> <a href="/wiki/%D0%AD%D0%BD%D0%B5%D0%BE%D0%BB%D0%B8%D1%82" class="mw-redirect" title="Энеолит">энеолитическая</a> культура 3000—2600 годов до&#160;н.&#160;э. Сложилась под сильным влиянием <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%B0%D1%86%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Костолацкая культура">костолацкой культуры</a>.
</p><p>Главные археологические находки были обнаружены в <a href="/wiki/%D0%A5%D0%BE%D1%80%D0%B2%D0%B0%D1%82%D0%B8%D1%8F" title="Хорватия">Хорватии</a> недалеко от города <a href="/wiki/%D0%92%D1%83%D0%BA%D0%BE%D0%B2%D0%B0%D1%80" title="Вуковар">Вуковар</a>, в том числе т.&#160;н. «вучедольская голубка»&#160;(<a href="/wiki/%D0%A5%D0%BE%D1%80%D0%B2%D0%B0%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Хорватский язык">хорв.</a> <span lang="hr" style="font-style:italic;">vučedolska golubica</span>) — керамический сосуд, украшенный различными орнаментами и выполнявший функцию <a href="/wiki/%D0%9A%D0%B0%D0%B4%D0%B8%D0%BB%D1%8C%D0%BD%D0%B8%D1%86%D0%B0" class="mw-redirect" title="Кадильница">кадильницы</a>. Сосуд был найден в <a href="/wiki/1938_%D0%B3%D0%BE%D0%B4" title="1938 год">1938 году</a>. Предположительно он был создан между 2800 и 2400 годами до нашей эры<sup id="cite_ref-MMB_1-0" class="reference"><a href="#cite_note-MMB-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>. В настоящее время «вучедольская голубка» находится в <a href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_(%D0%97%D0%B0%D0%B3%D1%80%D0%B5%D0%B1)" title="Археологический музей (Загреб)">Археологическом музее в Загребе</a>.
</p><p>Основу хозяйства вучедольцев составляло животноводство. Считается, что они практиковали <a href="/wiki/%D0%A7%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%B6%D0%B5%D1%80%D1%82%D0%B2%D0%BE%D0%BF%D1%80%D0%B8%D0%BD%D0%BE%D1%88%D0%B5%D0%BD%D0%B8%D0%B5" title="Человеческое жертвоприношение">человеческие жертвоприношения.</a> Хорватские учёные предполагают, что носители вучедольской культуры обладали древнейшим индоевропейским <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Календарь">календарём</a> и имели начальные астрономические знания. На керамическом сосуде из-под города <a href="/wiki/%D0%92%D0%B8%D0%BD%D0%BA%D0%BE%D0%B2%D1%86%D0%B8" title="Винковци">Винковцы</a> обнаружены (предположительно) изображения созвездий <a href="/wiki/%D0%9E%D1%80%D0%B8%D0%BE%D0%BD_(%D1%81%D0%BE%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B8%D0%B5)" title="Орион (созвездие)">Ориона</a>, <a href="/wiki/%D0%9A%D0%B0%D1%81%D1%81%D0%B8%D0%BE%D0%BF%D0%B5%D1%8F_(%D1%81%D0%BE%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B8%D0%B5)" title="Кассиопея (созвездие)">Кассиопеи</a>, <a href="/wiki/%D0%9B%D0%B5%D0%B1%D0%B5%D0%B4%D1%8C_(%D1%81%D0%BE%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B8%D0%B5)" title="Лебедь (созвездие)">Лебедя</a>, <a href="/wiki/%D0%91%D0%BB%D0%B8%D0%B7%D0%BD%D0%B5%D1%86%D1%8B_(%D1%81%D0%BE%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B8%D0%B5)" title="Близнецы (созвездие)">Близнецов</a>, <a href="/wiki/%D0%9F%D0%B5%D0%B3%D0%B0%D1%81_(%D1%81%D0%BE%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B8%D0%B5)" title="Пегас (созвездие)">Пегаса</a>, <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B5" title="Солнце">Солнца</a>, скопления <a href="/wiki/%D0%9F%D0%BB%D0%B5%D1%8F%D0%B4%D1%8B_(%D0%B7%D0%B2%D1%91%D0%B7%D0%B4%D0%BD%D0%BE%D0%B5_%D1%81%D0%BA%D0%BE%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)" title="Плеяды (звёздное скопление)">Плеяды</a><sup id="cite_ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>Сформировались на базе <a href="/wiki/%D0%91%D0%B0%D0%B4%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Баденская культура">баденской культуры</a> (через посредство промежуточной <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%B0%D1%86%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Костолацкая культура">культуры Костолац</a>). Вучедольской культуре наследует <a href="/wiki/%D0%92%D0%B8%D0%BD%D0%BA%D0%BE%D0%B2%D0%B0%D1%86%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Винковацкая культура">винковацкая культура</a>, через посредство которой вучедольцы связаны с формированием <a href="/wiki/%D0%98%D0%BB%D0%BB%D0%B8%D1%80%D0%B8%D0%B9%D1%86%D1%8B" title="Иллирийцы">иллирийских племён</a><sup id="cite_ref-4" class="reference"><a href="#cite_note-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>Согласно мнению <a href="/wiki/%D0%93%D0%B8%D0%BC%D0%B1%D1%83%D1%82%D0%B0%D1%81,_%D0%9C%D0%B0%D1%80%D0%B8%D1%8F" title="Гимбутас, Мария">Марии Гимбутас</a>, носителями этой культуры были мигрировавшие и осевшие в Европе индоевропейцы. <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B3%D0%B0%D0%B9%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D0%B8%D1%87" title="Монгайт, Александр Львович">Александр&#160;Монгайт</a> объединял вучедольскую культуру с <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B4%D0%B7%D0%B5%D0%B5_(%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0)" title="Мондзее (культура)">мондзейской культурой</a>. Также у вучедольцев наблюдается общность в материальной культуре с соседней <a href="/wiki/%D0%A0%D0%B5%D0%BC%D0%B5%D0%B4%D0%B5%D0%BB%D0%BB%D0%BE_(%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0)" title="Ремеделло (культура)">культурой Ремеделло</a> в Италии и, возможно, с <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B0%D0%B4%D0%B0_(%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0)" title="Полада (культура)">культурой Полада</a>. Гибридом вучедольской культуры и <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%B4%D0%B8%D1%86%D0%B8%D1%8F_%D0%BA%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D0%BB%D0%BE%D0%B2%D0%B8%D0%B4%D0%BD%D1%8B%D1%85_%D0%BA%D1%83%D0%B1%D0%BA%D0%BE%D0%B2" title="Традиция колоколовидных кубков">культуры колоколовидных кубков</a> была <a href="/wiki/%D0%9B%D1%8E%D0%B1%D0%BB%D1%8F%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Люблянская культура">люблянская культура</a>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Палеогенетика"><span id=".D0.9F.D0.B0.D0.BB.D0.B5.D0.BE.D0.B3.D0.B5.D0.BD.D0.B5.D1.82.D0.B8.D0.BA.D0.B0"></span>Палеогенетика</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%92%D1%83%D1%87%D0%B5%D0%B4%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Палеогенетика»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D1%83%D1%87%D0%B5%D0%B4%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0&amp;action=edit&amp;section=1" title="Редактировать код раздела «Палеогенетика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Исследованные геномы носителей культуры представляют собой смесь двух предковых компонентов&#160;— <a href="/wiki/%D0%A0%D0%B0%D0%BD%D0%BD%D0%B8%D0%B5_%D0%B5%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B7%D0%B5%D0%BC%D0%BB%D0%B5%D0%B4%D0%B5%D0%BB%D1%8C%D1%86%D1%8B" title="Ранние европейские земледельцы">ранних европейских земледельцев</a> и <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B5_%D1%81%D1%82%D0%B5%D0%BF%D0%BD%D1%8B%D0%B5_%D1%81%D0%BA%D0%BE%D1%82%D0%BE%D0%B2%D0%BE%D0%B4%D1%8B" title="Западные степные скотоводы">западных степных скотоводов</a> при незначительной примеси <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%BE%D0%B5%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%BE%D1%85%D0%BE%D1%82%D0%BD%D0%B8%D0%BA%D0%B8-%D1%81%D0%BE%D0%B1%D0%B8%D1%80%D0%B0%D1%82%D0%B5%D0%BB%D0%B8" title="Западноевропейские охотники-собиратели">западноевропейских охотников-собирателей</a><sup id="cite_ref-Patterson2022_5-0" class="reference"><a href="#cite_note-Patterson2022-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>У представителей вучедольской культуры были определены митохондриальные гаплогруппы <a href="/wiki/%D0%93%D0%B0%D0%BF%D0%BB%D0%BE%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_T2c2_(%D0%BC%D1%82%D0%94%D0%9D%D0%9A)" title="Гаплогруппа T2c2 (мтДНК)">T2c2</a>, <a href="/wiki/%D0%93%D0%B0%D0%BF%D0%BB%D0%BE%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_T2e_(%D0%BC%D1%82%D0%94%D0%9D%D0%9A)" title="Гаплогруппа T2e (мтДНК)">T2e</a>, <a href="/wiki/%D0%93%D0%B0%D0%BF%D0%BB%D0%BE%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_U4_(%D0%BC%D1%82%D0%94%D0%9D%D0%9A)" title="Гаплогруппа U4 (мтДНК)">U4a</a> и Y-хромосомные гаплогруппы <a href="/wiki/%D0%93%D0%B0%D0%BF%D0%BB%D0%BE%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_R-Z2103" title="Гаплогруппа R-Z2103">R1b1a1a2a2-CTS1078/Z2103</a> и <a href="/wiki/%D0%93%D0%B0%D0%BF%D0%BB%D0%BE%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_G2_(Y-%D0%94%D0%9D%D0%9A)" title="Гаплогруппа G2 (Y-ДНК)">G2a2a1a2a</a><sup id="cite_ref-IainM_6-0" class="reference"><a href="#cite_note-IainM-6"><span class="cite-bracket">&#91;</span>6<span class="cite-bracket">&#93;</span></a></sup>.
</p>
